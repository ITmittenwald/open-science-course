Docker

  • virtualization tool to wrap software with all its dependencies in one package (image, container)
  • enables to run many isolated computational environments
  • open source code and freemium platform
  • alternatives:
    • virtual machines
    • Vagrant
    • Singularity
    • ...


Docker logo

Environment specifications

Dockerfile: plain text specification of how the computational environment should look like in Docker 

FROM ubuntu:18.04
RUN apt-get update
RUN apt-get install -y \
        g++ \
        python \
        python-numpy

requirements.txt: list of Python packages (and optionally their versions) consumed by package installer 

pyunpack
GDAL==3.0.1

More on Binder: environment.yml, runtime.txt, apt.txt, DESCRIPTION, default.nix, postBuild, start, ...

Summary

  • GitHub hosts Git repositories and allows editing the content
  • Binder uses Docker to create the customized computational environments
  • JupyterLab runs and and enables editing of Python scripts and Jupter Notebooks
